## 创建管理员方式
由于本系统并没有考虑多人使用，因此也就没有开放用户注册功能，因此管理员要手动插入到数据库中，不过这里的密码是经过两次`md5`加密的，也就是说存到数据库中的密文是：
```
pass = md5(md5(pass))
```
`md5`加密的话网上有很多在线网站，这里就不演示了
